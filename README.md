## 개요
이 프로젝트는 SQL Server 데이터베이스의 모니터링을 위해 제작되었습니다.

서비스를 운영하는 중에 데이터베이스에서 문제가 발생될 경우 전체 시스템에 심각한 영향을 주게 되므로 보다 빠른 대처가 필요합니다. 따라서 운영 데이터베이스의 상태를 상시 모니터링하여 이상 증상이 발생될 경우 즉시 관리자에게 알림을 보내 빠른 대처가 가능하도록 하는 것이 무엇보다 중요합니다.

본 솔루션은 위와 같이 데이터베이스를 상시 모니터링하여 서비스를 항상 안전한 상태로 유지할 수 있도록 도움을 주기 위하여 제작되었습니다.
